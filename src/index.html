Datavisualisate

De visualisatie hiernaast maakt gebruik van live data uit het IoT stadslab. Deze data wordt opgehaald vanuit The Things Network en verzonden van de server via een websocket. Bij ieder TTN bericht event wordt vervolgens een nieuw punt in de grafiek getekend. De verschillende bericht die binnen komen vanuit de server zijn rechts onderin te zien.
